Configure the JSON below. If using personalize, you can optionally specify a Marketing Alias, to test the messaging. When ready, hit the Generate button.


Configuration

In the style object, specify the desired palette using one of the available keys in Palettes. Aditionally, specify the desired width for the widget.

Any and all sections are option. Simply delete an entire property and it's object to remove it from the configuration.

Enter any desired (and default) message text into the text property value. If you'd like to make that text hyperlinked, specify the full destination URL in the link property value; otherwise, leave that value as false.

Specific closeText can be applied. To use the default of "Close", simply remove that property.

Personalization

Set the personalization property to true to enable.

In the personalizations object, specificy the config section and the alternate text to use. Make sure to include ##marketingAlias## in the syntax, for Company Name to dynamically be inserted. If no valid Marketing Alias is found, the default text in for the section is used.
