Welcome to the Autodesk Agile Personalization Widget Configuration Wizard.

The widget is designed to strategically deliver targeted messaging on any Autodesk site. It floats into view, and won't get in the way.

This page is intended to help you quickly develop new personalization configurations for the widget, which can be deployed on any site leveraging the enterprise Tag Management System (Tealium) and the Web Analytics Foundation.

Configuration

In the style object, specify the desired palette using one of the available keys in Palettes. Aditionally, specify the desired width for the widget.

Any and all sections are optional. Simply delete an entire property and it's object to remove it from the widget.

Text & Links

Enter any desired (and default) message text into the text property values. To hyperlink text, specify the full destination URL in the corresponding link property value; otherwise, leave that value as false.

Specific closeText can be applied. To use the default of "Close", simply remove that property.

Personalization

To enable personalization, set the personalization property to true.

In the personalizations object, specificy the config section and the alternate text to use. Make sure to include ##marketingAlias## in the syntax, for Company Name to dynamically be inserted. If no valid Marketing Alias is found, the default text will be displayed.
